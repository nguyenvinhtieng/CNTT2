Hướng dẫn setup project
Yêu cầu chung: 
+ Máy có cài đặt NodeJS (Có thể cài đặt NodeJs tại: https://nodejs.org/en/download/)

- Backend:
    Yêu cầu: Máy có cài đặt MongoDb (Có thể cài đặt tại: https://www.mongodb.com/try/download/shell )
    Máy có kết nối internet(để kết nối tới các dịch vụ bên thứ ba như mega và cloudinary)
    Cách chạy:
        B1: Mở terminal (hoặc cmd) trỏ đến thư mục chứa source backend (là folder backend_express)
        B2: Chạy lệnh `npm install` và chờ cho quá trình cài đặt các pakage hoàn tất
        B3: Chạy lệnh `npm run dev` 
        Như vậy server sẽ chạy tại http://localhost:3001 theo như config của dự án 

- Frontend:
    Cách chạy: 
        B1: Mở terminal (hoặc cmd) trỏ đến thư mục chứa source frontend (là folder frontend)
        B2: Chạy lệnh `npm install` và chờ cho quá trình cài đặt các pakage hoàn tất
        B3: Chạy lệnh `npm run dev`
        Server fronend sẽ chạy tại http://localhost:3000

Mở trình duyệt và truy cập vào đường dẫn http://localhost:3000 để có thể truy cập và sử dụng trang web
